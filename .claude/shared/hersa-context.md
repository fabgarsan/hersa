**Hersa** is an end-to-end graduation event management platform for schools and institutions.

**Services:** photography (pre-graduation + ceremony-day), full graduation-day logistics, graduation packages (photos, dinners, parties), academic gown (toga) rental and sales, auditorium booking and management, masters of ceremony, graduation attire, and diplomas.

**Clients:** schools/institutions (B2B) and graduating students (B2C).

**Architecture:** monorepo with a React 19 + TypeScript frontend (SPA) consuming a Django + DRF REST API. Deployed on AWS (S3 + Elastic Beanstalk + RDS PostgreSQL).
